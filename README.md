# IELTS単語マスター - IELTS Vocabulary Builder

IELTS Band 6 を目指す学習者のための語彙学習アプリ。

## 特徴
- 📚 650+ IELTS Band 6 単語（10テーマ）
- 🎴 SRSフラッシュカード
- ❓ クイズモード
- 📝 穴埋め問題
- 📖 ストーリーラボ
- ✍️ ライティングチャレンジ
- 📊 進捗ダッシュボード
- 🏆 実績バッジ
- 📅 学習カレンダー
- 🔊 音声読み上げ（速度調整可能）
- 🖨️ 印刷エクスポート
- ⌨️ キーボードショートカット
- 🌙 ダークモード
- 🇯🇵 日本語UI + 英語表示
- 💾 データエクスポート/インポート
- 📱 PWA対応（ホーム画面に追加可能）

## インストール方法
1. このURLにアクセス
2. iOS: 共有ボタン → 「ホーム画面に追加」
3. オフラインでも動作！
